# Alphadoors

Smart Security & Access Solutions — React + TypeScript + Vite site deployed on GitHub Pages.

## Live Site

https://vasamsiddhartha.github.io/Alphadoors/

## Stack

- React 19 + TypeScript
- Vite (with `base: '/Alphadoors/'` for GitHub Pages)
- Framer Motion, Swiper, Lucide React
- GitHub Actions → GitHub Pages (automatic deploy on push to `main`)

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
